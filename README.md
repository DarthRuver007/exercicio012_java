# Programa para solicitar dois números inteiros e verificar qual é o número maior e mostrar na tela, ou se forem iguais, mostrar uma mensagem indicando esta informação.

Programa que solicita dois números inteiros, verifica qual é o número maior e mostra na tela, ou se forem iguais, mostra uma mensagem indicando esta informação.

Desenvolvido utilizado JDK 1.8 e NetBeans 13.
